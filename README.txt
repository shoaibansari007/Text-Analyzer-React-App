# Text Analyzer App

## Overview

Welcome to the Text Analyzer app, a simple React application designed to analyze text input. This project is ideal for those looking to get started with React and explore basic text manipulation features.

## Features

1. **Punctuation Calculation:**
   - Analyze the number of punctuation marks in the provided text.

2. **Sentence Calculation:**
   - Count the number of sentences in the input text.

3. **Words Calculation:**
   - Calculate the total number of words in the given text.

4. **Character Calculation:**
   - Determine the total character count, including spaces, in the entered text.

5. **Text Case Conversion:**
   - Convert the entire text to uppercase or lowercase as needed.
